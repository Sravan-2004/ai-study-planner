# 📚 AI Study Planner - Complete Study Buddy

An intelligent AI-powered study planner and study buddy application that helps students organize their learning, track progress, and get personalized study recommendations.

## ✨ Features

- **📅 Study Schedule Generation** - AI generates personalized study timetables
- **✅ Task Tracking** - Manage and track all study tasks with priorities
- **📝 Quiz Generation** - AI-generated practice questions by subject
- **💬 AI Study Buddy** - Chat with AI for explanations and study support
- **📊 Progress Tracking** - Visualize your learning journey with analytics
- **🎯 Personalized Recommendations** - Get smart study suggestions
- **⏰ Reminders & Notifications** - Real-time notifications via Socket.io
- **📱 Responsive Dashboard** - Beautiful, modern UI with Tailwind CSS

## 🛠️ Tech Stack

### Backend
- **Node.js + Express** - Server framework
- **TypeScript** - Type safety
- **MongoDB** - Database
- **OpenAI API** - AI/LLM capabilities
- **Socket.io** - Real-time notifications
- **JWT** - Authentication

### Frontend
- **React + TypeScript** - UI library
- **Tailwind CSS** - Styling
- **Zustand** - State management
- **Recharts** - Data visualization
- **Axios** - HTTP client
- **React Router** - Navigation
